About Me

Recently graduated from the University of Regina with a Master of Science (Cooperative education) degree in Computer Science. I am a passionate software developer with experience in agile-based software development for business-oriented applications. Currently working in a team for the University of Regina, developing a Full Stack Android application for digitalizing paper-based safety procedures. The application is developed using Kotlin, MySQL database is used as a backend server which is connected to the application by a REST API developed using Python-based Flask to provide database connectivity. I have worked on various machine learning projects wherein data science concepts and algorithms were implemented. Developed an Article Activity visualization dashboard where text summarization, keyword extraction, and sentiment analysis was done on 1 million news articles. The dashboard was developed using D3 JavaScript and data mining was done using Python. Implemented a Maximum Likelihood Estimation model to predict the probability of default for a financial institution. I have done association rule mining using apriori algorithm on supermarket transaction data where I found multiple rules for items/products which are purchased together. Developed a rain prediction model using the ensemble learning model.

I have worked with ExxonMobil where I was part of the Data Science team to create custom Machine Learning models. I also supported the business by providing SharePoint and Nintex based web applications using agile development practices. I also solved business problems by understanding their day to day job responsibilities and incorporating data science products.
